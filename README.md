# KEYB0ARD
**Minimalist Mac Styled Keyboard!**

A compact, minimalist styled keyboard PCB

 - Embedded LCD Display
   
 - Weather, Time, Battery percentage etc.
   
 - Created in KiCad 10.0
   
 - PCB design
   
 - Run with a Pico Zero 2 W
   
 - No Backlights

**90 keys!**

- 13 Functions

- 1 Shortcut Key

- Power Button
